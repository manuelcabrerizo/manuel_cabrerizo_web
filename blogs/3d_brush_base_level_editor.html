This project is a 3D level editor inspired by the Hammer editor, designed to provide an intuitive and versatile tool for creating detailed 3D environments.

Brush-Based Level Editor

This editor uses a brush-based approach, meaning that objects are created as "brushes." A brush is a convex volume formed by the intersection of multiple planes. For example, a cube is considered a brush because it can be defined by the intersection of six planes that enclose a volume.

The brush-based system allows for greater flexibility and precision when creating and manipulating 3D geometry, making it ideal for level design in games or simulations.

Key features of the editor include:

  • Add and remove brushes to build or modify the 3D environment
  • Apply and remove textures to customize the appearance of each brush
  • Paint individual brushes with different textures for visual variety and depth
  • Clip brushes to create more complex shapes and architectural details
  • Save and load scenes or models as binary files for easy editing and reuse
  • Multiple viewports (top, front, and side) to aid in precise positioning and alignment of brushes
