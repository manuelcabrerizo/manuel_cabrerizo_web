This project is a 3D level editor inspired by the Hammer editor, designed to provide an intuitive and versatile tool for creating detailed 3D environments. The editor combines flexibility with ease of use, allowing level designers to create, modify, and experiment with 3D worlds in an interactive and efficient way. Whether for games, simulations, or architectural visualizations, this tool aims to empower users to bring their creative ideas to life.

Brush-Based Level Editor

The core of this editor is its brush-based approach, where 3D objects are constructed as "brushes." A brush is essentially a convex volume defined by the intersection of multiple planes. For example, a cube is considered a brush because it can be described by six planes that enclose a rectangular volume. This method allows for a modular, highly flexible approach to 3D object creation.

The brush-based system gives users the freedom to build levels by combining simple geometric shapes into more complex structures, offering a fast way to create environments. Additionally, the system supports precision, as brushes can be manipulated and clipped in various ways, making it ideal for creating intricate 3D worlds.

Key features of the editor include:

  • Add and remove brushes: This feature enables users to easily build or modify the 3D environment by adding or removing geometric elements. Each brush represents a specific piece of geometry, such as a wall, floor, or any other structure, and can be positioned and adjusted to create diverse environments.
  • Apply and remove textures: The ability to apply textures to brushes allows for aesthetic customization. Users can change the look of objects, creating realistic surfaces or stylized effects. Textures can be easily added or removed to suit different design requirements, enabling rapid iteration and experimentation.
  • Clip brushes: This feature allows users to clip brushes, cutting them into more complex shapes. Clipping makes it possible to design highly detailed structures and terrain features, such as windows, doors, or irregularly shaped rooms. The brush clipping functionality gives the editor its power, enabling designers to create everything from simple objects to intricate geometric configurations.
  • Save and load scenes or models: The editor allows users to save and load entire scenes or individual models in binary format. This feature simplifies the editing process, as designers can store and recall their work quickly, facilitating collaboration and version control. Scenes can be saved for later editing or exported to be used in other software or engines.
  • Multiple viewports (top, front, and side): This feature offers different perspectives on the scene, making it easier to position and align brushes precisely. By switching between top, front, and side viewports, users can get a clearer idea of the geometry they are working with, ensuring greater accuracy when creating and modifying the environment.
