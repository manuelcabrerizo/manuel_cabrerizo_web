3D Animation, Software Renderer, and Physics Simulation

Sample Instructions

  • W A S D keys move player
  • Arrow keys applay forces to the sphere
  • Xbox joystick support for moving the player with the left stick

This project explores multiple advanced areas, acting as a testing ground for three main components:

Software Renderer

I developed a software renderer from scratch as an exercise in learning. Building it from the ground up allowed me to understand every step involved in rendering, from drawing triangles on the screen to eventually animating 3D models.

Key features of the renderer include:

  • Rendering triangles and animated models on the screen
  • Drawing lines and setting world, view, and projection matrices
  • Rendering both static and animated 3D models
  • Applying textures using UV coordinates
  • Clipping triangles outside the view frustum
  • Using AVX instructions for efficient triangle rasterization, painting 8 pixels at a time

3D Animation and Interpolation

This project also includes a skeletal animation system, which interpolates between different poses of a skeleton to animate a 3D model. This allows smooth transitions between animations and brings models to life.

Key features of the animation system:

  • Animating complex 3D models
  • Blending multiple animations to create smooth, realistic transitions
  • Loading models and animation data using Assimp

Physics Simulation

The physics component comprises two systems: particle simulation and rigid body simulation. These simulations add realism and interactivity to 3D environments.

Key features of the physics simulation:

  • Simulating particle movement with forces and constraints
  • Creating particle systems connected by springs
  • Simulating rigid body dynamics, including spheres and boxes
  • Handling collision detection and resolution for spheres and boxes
