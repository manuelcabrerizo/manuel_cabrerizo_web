Software Renderer

I developed a software renderer from scratch as an exercise in learning. Building it from the ground up allowed me to understand every step involved in rendering, from drawing triangles on the screen to eventually animating 3D models.

Key features of the renderer include:

  • Rendering triangles and animated models on the screen
  • Drawing lines and setting world, view, and projection matrices
  • Rendering both static and animated 3D models
  • Applying textures using UV coordinates
  • Clipping triangles outside the view frustum
  • Using AVX instructions for efficient triangle rasterization, painting 8 pixels at a time
